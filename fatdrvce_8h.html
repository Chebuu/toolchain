uint8_t fat_Init ( void  )

Initializes the FAT filesystem and allows other FAT functions to be used. Before calling this function, you must use fat_Find and fat_Select in order to locate the correct FAT partition.

Warning
When using the FAT library, calls to os_GetCSC() will cause USB not to work. This will probably be fixed in later revisions, but just use the keypadc library for now.
Returns
0 on success.
