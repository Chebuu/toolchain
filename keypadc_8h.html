KB_MODE_1_PRESS 

Interrupt set when a key is pressed in MODE_1_INDISCRIMINATE

enum kb_scan_mode_t

Different available scanning modes

Enumerator:
MODE_0_IDLE 

Keypad scanning is idle

MODE_1_INDISCRIMINATE 

Indiscriminate key detection. Data registers are invalid, but whenever any key is pressed, interrupt KB_MODE_1_PRESS is set (and cannot be cleared until the key is released).

MODE_2_SINGLE 

Single scan. The keypad is scanned once, and then the mode returns to MODE_0_IDLE.

MODE_3_CONTINUOUS 

Continuous scan. When scanning completes, it just starts over again after a delay.
